# Knowledge Base

This folder stores validated business analysis outputs created by BIVA Agent and future StarStart agents.

## Sections

| Folder | Prefix | Purpose |
| --- | --- | --- |
| `01_Idea_Intake` | `IDEA` | Raw ideas, structured intake, assumptions, and initial notes. |
| `02_Feasibility` | `FEAS` | Technical, operational, legal, and execution feasibility analysis. |
| `03_Market_Research` | `MKT` | Market context, customer segments, competitors, and demand signals. |
| `04_Validation_Plans` | `VAL` | Experiments, MVP tests, success metrics, and decision rules. |
| `05_Business_Cases` | `CASE` | Business model, economics, strategic rationale, and investment logic. |
| `06_Decision_Logs` | `DEC` | GO, REVISE, HOLD, and NO-GO decisions with evidence history. |

## Storage Rule

Use `docs/NAMING_CONVENTION.md` for every file stored here. Every document should include metadata, confidence level, related files, and decision status.
