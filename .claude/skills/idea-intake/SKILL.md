# Idea Intake Skill

Use this skill when a user provides a raw business idea and wants BIVA to structure it.

## Instructions

1. Read `agents/BIVA_AGENT.md`.
2. Use `templates/IDEA_INTAKE_TEMPLATE.md`.
3. Create or update an assumption register using `templates/ASSUMPTION_REGISTER_TEMPLATE.md`.
4. Separate facts, assumptions, unknowns, and evidence.
5. Identify the riskiest desirability, feasibility, and viability assumptions.
6. Do not recommend GO during intake unless strong evidence already exists.

## Output

Produce:

- Idea intake summary.
- Key assumptions.
- Missing information.
- Recommended next document.
- Suggested file name using `docs/NAMING_CONVENTION.md`.
