# Validation Plan Skill

Use this skill when BIVA needs to design experiments before a product is built.

## Instructions

1. Use `templates/VALIDATION_PLAN_TEMPLATE.md`.
2. Use `templates/ASSUMPTION_REGISTER_TEMPLATE.md` to rank assumptions.
3. Use `frameworks/LEAN_STARTUP.md` for experiment design.
4. Use `frameworks/EVIDENCE_LADDER.md` to define success thresholds.
5. Test the riskiest assumption first.
6. Prefer low-cost behavioral tests over feature-heavy MVPs.

## Output

Every experiment must include:

- Hypothesis.
- Method.
- Audience.
- Timeline.
- Success metric.
- Evidence level expected.
- Decision rule.
