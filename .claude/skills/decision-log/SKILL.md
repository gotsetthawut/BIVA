# Decision Log Skill

Use this skill when BIVA needs to record GO, REVISE, HOLD, or NO-GO decisions.

## Instructions

1. Use `templates/DECISION_LOG_TEMPLATE.md`.
2. Review the evidence log before deciding.
3. Use `frameworks/STAGE_GATE.md`.
4. Require an Evidence Ladder rating.
5. Preserve uncertainty and review triggers.
6. Do not overwrite old decisions. Create a new decision log when evidence changes.

## Output

Include:

- Decision.
- Evidence reviewed.
- Assumptions supported or invalidated.
- Risks considered.
- Rationale.
- Next actions.
- Review trigger.
