# Value Proposition Canvas

## Purpose

The Value Proposition Canvas helps BIVA test whether the proposed solution fits a real customer job, pain, or desired gain.

## Customer Profile

| Area | Description |
| --- | --- |
| Customer Jobs | What the customer is trying to accomplish |
| Pains | Costs, frustrations, risks, or blockers |
| Gains | Desired outcomes, improvements, or benefits |

## Value Map

| Area | Description |
| --- | --- |
| Products and Services | What is being offered |
| Pain Relievers | How the offer reduces pains |
| Gain Creators | How the offer creates desired gains |

## Fit Questions

- Which customer job is most urgent?
- Which pain is frequent, costly, or risky?
- Which gain is valuable enough to change behavior?
- Does the proposed solution address the highest-priority pain?
- What proof would show fit?

## BIVA Usage

Use this framework when an idea is broad, when customer language is vague, or when the value proposition needs to be sharpened before validation.

## Output Format

```markdown
## Value Proposition Canvas

### Customer Jobs

### Pains

### Gains

### Products and Services

### Gain Creators

### Fit Assessment
```
